# Lists in Python

Lists are mutable ordered collections used to store sequences of items.

## Creating a List
```python
l = [1, 2, 3]
```

## Common Methods
- `append(x)`
- `extend(iterable)`
- `insert(i, x)`
- `remove(x)`
- `pop([i])`
- `sort()`, `reverse()`

## Use Cases
- Dynamic collections
- Indexed data storage
- Iterative operations
